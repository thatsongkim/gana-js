# gana-js

## what is this?

simple node app for learning japanese hiragana and katakana.

## why...???

i wanted to learn gana at a low cost.

## low cost? really?

idk, but it was fun.

## how to use?

### commands

- on start
    - select hiragana: `hi`
    - select katakana: `ga`
- on question
  - input pronunciation as english. for example, `„Å≤` for `hi`
  - you can still input command
-settings
  - toggle sound: `set sound` or `set s` (default is off)
